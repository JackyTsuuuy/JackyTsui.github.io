面向对象相关BIF

1.issubclass(class,classinfo):第一个参数class是第二个classinfo的子类返回true

2.isinstance(object,classinfo):检查第一个参数object是否是第二个classinfo的对象，返回true或false

3.hasattr(object,name):检查第一个参数object对象是否含有第二个name的属性

4.getattr(object,name[,default]):返回对象指定的属性值

5.setattr(object,name,value)：设置指定属性的值
6.delattr(object,name):删除指定属性的值

7.property(fget=None,fset=NOne,fdel=None,doc=None):第一个参数是获得属性的方法，第二个是设置属性的方法，第三个是删除属性的方法

魔法方法：定制序列

定制容器不可变，只需定义len()和getitem()
定制容器可变，除len()和getitem()，还需定义setitem()和delitem()

len(self):返回容器中元素的个数
getitem(self,key)：获取容器中指定元素的行为，self[key]
setitem(self,key,value):设置容器中指定元素的行为，self[key]=value
delitem(self,key):删除容器中指定元素的行为，del self[key]

e.g.

c = Countlist(1,2,3,4)
c2 = Countlist(2,4,6,8)
