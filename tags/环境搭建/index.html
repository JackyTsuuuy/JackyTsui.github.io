这里所有示例机器在虚拟机环境下进行实验。
实验域环境的简单说明：

域控：
Ip:10.10.133.1
操作系统：win2008英文版
域名：test.com
客户机：
Pc1：
Ip:10.10.133.2
操作系统：win7
域账号：joom（普通域用户组）
Pc2：

域账号：boom（域管理员组）

客户机加入域：

这里以win7客户机为例

2.打开“高级系统设置”


3.在“计算机名”标签下选择“更改”


4.填入域名，这里以test.com为例，之后会弹出一个登录框，填写域账号和密码即可登录，这里域账号以joom为例。

Win2008(英文版)搭建简单域环境(2)

2016-04-01
  • 基础
  • 环境搭建

这里所有示例机器在虚拟机环境下进行实验。
实验域环境的简单说明：

域控：
Ip:10.10.133.1
操作系统：win2008英文版
域名：test.com
客户机：
Pc1：
Ip:10.10.133.2
操作系统：win7
域账号：joom（普通域用户组）
Pc2：
Ip:10.10.133.3
操作系统：winxp
域账号：boom（域管理员组）

域账号的配置：

一．图形界面方式:

  1. 登录域控
  2. 在”administrative Tools”中打开”Active Directory Domains and Trusts “
     

    3.选择刚建立的test.com，右键选择”Manage”


4.在”users”上右键，选择”New”下的”user”

 

5.填写要添加的账号信息，这里账户名以”joom”为例。同理，添加”boom”

6.填写密码，勾选”User cannot change password”用户不可改变密码，”password never expires”密码不过期，然后Next。然后Finish

7.然后Finish


8.同理添加boom账号，之后将boom账号添加入管理员组。在boom账号上右击，选择”properties”，打开”Member of”标签，打开”Add”，填写要加入的组” Domain Admins”,然后”OK”


9.查看添加好的信息

二．命令行方式

打开cmd输入命令：
1.命令格式
net user username password /add /domain

2.以用户名为boom ，密码为password为例
net user boom password /add /domain


如果提示这个则说明密码强度不够，可以换个长点的密码试试


3.将用户加入管理员组
net group “domain admins” boom /add /domain
4.完成，同理添加其他账户

Win2008(英文版)搭建简单域环境(1)

2016-04-01
  • 基础
  • 环境搭建

这里所有示例机器在虚拟机环境下进行实验。
实验域环境的简单说明：

域控：
Ip:10.10.133.1
操作系统：win2008英文版
域名：test.com
客户机：
Pc1：
Ip:10.10.133.2
操作系统：win7
域账号：joom（普通域用户组）
Pc2：
Ip:10.10.133.3
操作系统：winxp
域账号：boom（域管理员组）

域控的搭建：

首先设置网卡为NAT，ip为静态10.10.133.1，网关和DNS都配置为10.10.133.1

1.win+r，输入“dcpromo”，打开AD安装向导,next


2.next


3.选择”create a new domain in a new forest”,next


4.提示这个,表明要设置本地账户的密码


5.填写林根域的FQDN名(FQDN of the forest root domain),这里以test.com为例


6.选择林功能级别(Forest function level)，和域功能级别(Domain function level)，以下为支持情况，这里选择win2003为例



7.在“additional domain controller options”中选择”DNS server”,next


8.设置数据库，日志文件，SYSVOL的存储位置，这里使用默认位置


9.设置还原模式(Restore Mode)的密码


10.next

 
11.完成，之后选择重启
