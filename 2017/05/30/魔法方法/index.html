魔法方法

init(self[,…])
对一个对象进行初始化，返回none

new(cls[,…])
对象实例化时调用的第一个方法，并非init,第一个参数不是self，而是class，后面的参数会原封不动传给init。一般只有在继承一个不可变类型，但又需对其进行修改时使用，比如str->大写的str。

class C:
def init(self):
print(“init has been used”)
def del(self):
print(“del has been used”)

c1 = C()
init has been used
c2 = c1
c3 = c2
del c2
del c3
del c1
del has been used
