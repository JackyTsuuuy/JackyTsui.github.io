函数

def function(): print()

调用时会从当前代码往上找

函数文档：
格式：函数名.doc

e.g
显示print函数的函数文档

形参
实参
收集参数：参数前加号：param

def test(*params):
print(len(params))
print(params[1])

test(1,aa,3)

如print方法中的参数，本身就是收集参数

返回值：
一般来说，有返回值是函数，没返回值是过程

但python严格来讲只有函数，在不写返回值的时候返回None
可返回多个值，比如用一个列表：
return[1,’a’,2]
直接用逗号分隔，会返回元组：
return 1，2，3

全局变量:
作用于整个文件
只可在函数内读取
若在某函数内修改全局变量相当于在局部重新创建了一个同名局部变量
若在函数内修改全局变量，需用globle定义：

内嵌函数(内部函数)：
函数内创建另一个函数

闭包:
在一个内部函数里对外部作用域（非全局）的变量进行了引用，该内部函数就叫闭包
