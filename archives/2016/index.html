Win2008(英文版)搭建简单域环境(2)

2016-04-01
  • 基础
  • 环境搭建

这里所有示例机器在虚拟机环境下进行实验。
实验域环境的简单说明：

域控：
Ip:10.10.133.1
操作系统：win2008英文版
域名：test.com
客户机：
Pc1：
Ip:10.10.133.2
操作系统：win7
域账号：joom（普通域用户组）
Pc2：
Ip:10.10.133.3
操作系统：winxp
域账号：boom（域管理员组）

域账号的配置：

一．图形界面方式:

  1. 登录域控
  2. 在”administrative Tools”中打开”Active Directory Domains and Trusts “
     

    3.选择刚建立的test.com，右键选择”Manage”


4.在”users”上右键，选择”New”下的”user”

 

5.填写要添加的账号信息，这里账户名以”joom”为例。同理，添加”boom”

6.填写密码，勾选”User cannot change password”用户不可改变密码，”password never expires”密码不过期，然后Next。然后Finish

7.然后Finish


8.同理添加boom账号，之后将boom账号添加入管理员组。在boom账号上右击，选择”properties”，打开”Member of”标签，打开”Add”，填写要加入的组” Domain Admins”,然后”OK”


9.查看添加好的信息

二．命令行方式

打开cmd输入命令：
1.命令格式
net user username password /add /domain

2.以用户名为boom ，密码为password为例
net user boom password /add /domain


如果提示这个则说明密码强度不够，可以换个长点的密码试试


3.将用户加入管理员组
net group “domain admins” boom /add /domain
4.完成，同理添加其他账户

2016-03-31
  • 基础
  • git

背景：

最近老想着折腾一个属于自己的博客。(虽然简书用起来方便，但是github逼格高啊！)所以出于种种原因吧，还是觉得github能比较个人特色鲜明点。而且在有些国内的博客都要翻墙访问的情况下，github的个人主页居然不需要，这种事也让我感受到了意外的惊喜。所以反正都是要翻墙折腾，不如就选一个有逼格的，当然装逼也是得付出点代价的，需要稍稍花点时间来折腾。

因为刚开始折腾，所以本篇先主要讲如何将这些基本部件安装搭起来。
